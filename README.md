  <h3 align="center">DeginxPaste(Dpaste++)</h3>
  <p align="center">
    一个基于C++、QT和Mysql的多平台同步剪贴板工具，项目于2020年初进行开发，后面因为高中学业紧张而不得不停止开发，仅局限于Mac OS上开发。当时这个项目很有前瞻性的，那时候Windows还没有剪贴板，可惜没有继续开发完☹️，有兴趣的小伙伴们可以继续开发~
